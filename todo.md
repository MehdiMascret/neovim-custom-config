# GIT
## Commande Ex + Keymap 
```sh
:g (git)
:ga (git add)
:gi (git init)
:gc ... (git commit -m "...")
:gca (git commit --amend)
:gcan (git commit --amend --no-edit)
:<gc{options}>p (git commit with push directly after)
:gs ... (git switch ...)
```
## Neotree
'g' on neotree allow you to show diff between diff version.
